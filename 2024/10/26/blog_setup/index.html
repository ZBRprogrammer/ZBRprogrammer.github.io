The step to build up a blog

Created2024-10-26|Updated2024-10-26
|Post Views:

你需要的：

一个 Github 账号

Node.js（自行下载安装）

Git（自行下载安装）

Hexo

一个域名(Unnecessary)

一个云服务器(Unnecessary)

在 Github 上创建一个 repo ，命名为 <Your Username>.github.io ，勾选添加 Readme.md

选择一个空目录，打开cmd，运行

  • npm install hexo

  • hexo init

  • npm i

  • npm install hexo-deployer-git --save

修改生成文件中的_config.yml

deploy:

  type: ‘’

更改为

deploy:

  type: git

  repository: <你的仓库所在地址>

  branch: main

在cmd中，运行

  • hexo cl

  • hexo g

  • hexo d

现在可以在https://<Your Username>.github.io看到你的Blog了。
